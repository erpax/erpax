# standard

Implemented via the skills, never bespoke: each skill is the answer-path holding a standard's form — apply [[accounting]] for IFRS/SOX double-entry, [[currency]] for ISO-4217, [[identity]] for the audit chain, [[port]] for strict-standards. Every `@standard`/`@audit`/`@compliance` banner must be TRUE; a claimed-but-unimplemented standard is an [[aura]] gap. The [[sequence]] keeps many standards mutually compliant — apply the skill, the standard follows.
