# value

Use when a field carries a numeric or quantified substance — quantity value (paired with unit), exchange rate value, percentage, index. Generic numeric container; pairs with unit (measure), currency, or rate context to form money/quantity/rate.

Composes: [[measure]] · [[currency]] · [[rate]] · [[fields]] · [[calculate]].

**Law — [[law]]: value is a bare numeric substance that means nothing alone — it pairs with a context ([[measure]] unit, [[currency]], or [[rate]]) to become money/quantity/rate; the unit is never baked into the number.**
