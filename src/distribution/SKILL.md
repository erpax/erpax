# distribution

Use when analyzing data spread — normal/Poisson/exponential distributions, distribution testing (Shapiro-Wilk, Kolmogorov-Smirnov), distribution fitting, distribution-aware statistics (mean vs median), distribution plots/histograms.

Composes: [[calculate]] · [[aggregation]] · [[metric]].

**Law — [[law]]: a distribution is the SHAPE of data spread (normal·Poisson·exponential), and the shape dictates the honest statistic — mean vs median, which test (Shapiro-Wilk·KS) — so distribution-aware analysis precedes any summary.**
