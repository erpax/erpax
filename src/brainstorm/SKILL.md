# brainstorm — coordinated divergence ⊕ convergence

Brainstorming is the [[quantum]] solve run on ideation: **diverge** into a superposition of candidate ideas, then **converge** by collapse. It is a coordinated action — never one mind in series, but a [[team]] fanned out in parallel ([[society]] convened), each agent a distinct [[perspective]] ([[critic]] ⊕ [[artist]] — risk-first ⊕ user-first ⊕ MVP-first), so the idea-space is explored at once, not branch by branch.

## The two strokes

1. **Diverge (the [[breath]] in)** — fan out; every [[perspective]] generates freely, no judging yet. Each idea is a candidate [[atom]] — a not-yet-minted [[word]], an [[aura]] gap to fill. Breadth over depth; more amplitude is better.
2. **Converge (the breath out)** — [[merge]] the candidates: same idea ⇒ same content-[[uuid]] ⇒ one (dedup is free). The [[critic]] panel amplifies the survivors (judge by distinct lenses, a majority keeps it); the rest decohere. What survives is minted ([[generate]]).

Coordination is what makes it quantum rather than noisy: the agents are blind to each other while diverging (true superposition), and only [[merge]] at convergence — so collision is creation, not conflict. The matter-twin is a fan-out → merge → critic-panel workflow — the same loop the coordinated researchers and the audit ran, pointed at ideas.
