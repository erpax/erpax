# descent — horo position 7 (the wave returns)

`descent` is the **turn** of the [[horo]] ring — digit `7`, where the doubling [[coil]] folds back from the peak (`8 → 16 ⇒ 7`) and the **return** arc begins (`7·5·9`). The built flow now moves outward: inventory *shipped*, value *given*, the entry *posted out*. It is the [[give]] side of a [[flow]] — the harvest of what the ascent built — and the [[sequence]] anchor `7` ([[api]], where the architecture binds and surfaces). Between [[crest]] and [[round]].

**Law — [[law]]: [[horo]] position 7 — where the built flow folds back from the peak and moves outward (shipped · [[give|given]] · posted out), the first step of the return arc.**
