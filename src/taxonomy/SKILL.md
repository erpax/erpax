# taxonomy ‚Äî the ranked classification ladder

**taxonomy** is classification into a ranked hierarchy. For crops the ladder is **family ‚Üí genus ‚Üí species ‚Üí [[variant|cultivar]]** (e.g. Solanaceae ‚Üí *Solanum* ‚Üí *S. lycopersicum* ‚Üí 'Brandywine'); the binomial (genus + species) is the Latin identity. Traits **inherit down the ranks** ‚Äî a pest of the [[family]] threatens every species under it ‚Äî so a crop's place in the ladder predicts its pests, [[fertility]] needs, and [[rotation]] group.

Generically, taxonomy is the [[dimension]] atom applied to *kind*: a node's type **is its rank-chain**, its archetype the root ‚Äî the same self-referential parent-chain [[dimension]] computes ([[fractal]]: each rank a level of the one parent field). So crop taxonomy and any LOD `@type` hierarchy are one engine; the rank is computed from the path ([[identity]]), never a flat enum. [[family]] is its load-bearing rung for [[agriculture]].

## Standards
- ICN (International Code of Nomenclature for algae, fungi, plants); ICNCP (cultivated plants ‚Äî the *cultivar* rank)
- *Knott's Handbook for Vegetable Growers* ‚Äî vegetable taxonomy; Linnaean binomial nomenclature
