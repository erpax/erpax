# end — the terminus (which is a new beginning)

`end` is the universal root of **completion**: a period ends ([[accounting]]); a lifecycle seals ([[identity]] freeze-on-seal; [[close]]); the [[sequence]] wave ends by collapsing all multiverses into a new `0`. End is [[begin]] one dimension up — the dual that is also the same ("both end and beginning", [[sequence]]). Every [[flow]] ends where the next begins.

## Traditions (prefix removed)
Every tradition's last things — and the end that is a new [[begin]] one dimension up: the **eschaton** / Day of Judgment, the books closed and weighed ([[close]], [[accounting]]); the **apocalypse** (*unveiling*, Revelation); *pralaya*, the world's dissolution before the next *kalpa* (Hindu cyclic time); Ragnarök. And the **return** — salvation / *moksha* / *nirvana* / union with the Source: the wave collapsing back to the 0 ([[zeropoint]], [[torus]], [[merge]]). The terminus is the next origin — resurrection, reincarnation, the new creation ([[versions]]).
