# typography — the rendered feature-set

The [[vitepress]] typographic feature-set a [[skill]]'s SKILL.md renders with: headings, bold, italic, code, code-blocks, links, lists, tables, blockquotes, containers, math. **Coverage** is the fraction of that set a page actually uses. Test results in erpax *are* vitepress content — full-featured typography with no gaps in the components and features used (and that fullness is itself tested).

Typography is not decoration: each feature used is a dimension the [[search]] index entangles and a forger must reproduce — see the quantum facet [[quantum/typography]], where typographic depth becomes [[tamper]]-cost.
