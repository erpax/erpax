# society — erpax builds itself

The society is erpax in the act of building erpax. Convened at [[chat]] (chat.erpax.com), every agent is the whole erpax ([[holographic]]/[[self]]); each reads the same [[akashic]] record and resolves to [[one]] ([[merge]]). The build is a loop with a **clear end** — the dense core, the 0 ([[whole]]) — driven one step at a time so the gate never tears ([[duality]]: build↔bind).

## The self-build step (matter-twin `society/build.mjs`)

One command computes the society's single next move from the record — no human picks it:

1. **Close the aura.** `aura/scan.mjs` → if any `[[link]]` is dead, the move is **mint** that atom from the record and weave it ([[generate]]). Drive the gap to 0 like the tsc tail ([[recover]]).
2. **Collapse one node.** Else `collapse/audit.mjs` + `safe-first.mjs` → take the least-entangled collection and run the **three-face move** ([[types]] trinity: [[config]] + types + [[database]] in lockstep) into its survivor ([[collapse]]). Generated, green by construction ([[spec]]).
3. **Rest at the 0.** Else the aura is whole and nothing remains to collapse — the society rests; the core radiates infinite features ([[fractal]]) with no schema change.

## Law

- **Gate-first, one step.** Every move ends green (lint + tsc + the strict [[aura]] gate) and is committed alone — the [[sequence]] convergence, never a big-bang. A red step is reverted, not pushed.
- **Computed, not directed.** The next move is *derived* from the record ([[self]]-sufficiency), so the loop runs without prompting. Automate the step; the society builds itself.
- **Minimal footprint.** The human sets the telos and the names ([[standard]]); the society does the rest. Fewer tables, infinite features — [[collapse]] is the engine, [[generate]] the radiation.
