# vocabulary — words only from the shared scheme

Whatever the language, the WORDS are written only from one shared vocabulary. The scheme (`./words.ts`, generated by `scripts/vocab-gen.mjs`) is grounded in standards: the English dictionary (∪ inflections), **schema.org** (the `@schema.org component word` atoms — the standard carries even its own spellings, so `clincal`/`biomechnical` are grounded, not typos), **standard codes** (numeric ISO/EN versions, abbreviations), and a curated set of established **domain** terms. An [[atom]] is GROUNDED iff it is a vocabulary root or COMPOSES from roots (`budgetvariance` → `budget` + `variance`).

An ungrounded word is [[entropy]] — and a typo, an ad-hoc invention, and a forgery are the same thing: a word from outside the scheme. So the gate is a [[tamper]]-cost lever ([[unavoidable]]): every word wired to the shared vocabulary is one more dimension a forger must satisfy. `audit()` drives that entropy to ZERO; the corpus is 0-entropy today (all 2241 atoms grounded).

**The errors compute the solution.** When a stray slips in, `./test.ts` does not merely flag it — `suggest()` computes the zero-entropy fix: the nearest vocabulary word by edit distance (a typo is one edit from the scheme), or the decomposition into roots (`path=type`), or "extend the vocabulary." The fix that restores 0 entropy is read straight off the failure.

This is the [[merge]] law at the word level (same word = same root = same uuid) and the sibling of md-purity ([[trinity]]) and the [[standards]] scheme. New words must extend `words.ts` deliberately — never invented in place.

Matter-twin: `src/vocabulary/index.ts`. Composes [[entropy]] · [[tamper]] · [[unavoidable]] · [[merge]] · [[trinity]] · [[standards]] · [[word]] · [[collapse]] · [[matrix]].
