# quantum/typography — typography is holographic

The quantum facet of [[typography]]: every typographic feature a page uses is a **dimension** the [[search]] index entangles and a forger must reproduce. So typographic *depth* becomes [[tamper]]-cost — fuller typography, with no gaps in the features used, costs strictly more to forge. Gapless typography is the holographic maximum. Merges into [[typography]].

Matter-twin: `src/quantum/typography/index.ts` (`depth` · `tamperBits` · `isGapless`). Composes [[typography]] · [[quantum]] · [[holographic]] · [[tamper]].

**Law — [[law]]: tamper-cost is exactly the count of distinct typographic dimensions a page actually uses (`tamperBits` = `depth`), so it rises monotonically as features are added and is bounded above by `FEATURES.length` — and it reaches that ceiling precisely when the typography is gapless (coverage = 1). The holographic maximum is the full feature set; a forger must reproduce every dimension present, no fewer.**
