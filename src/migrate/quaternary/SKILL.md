# quaternary

The four-file law: every atom folder is exactly **{SKILL.md · index.ts · test.ts · translations.ts}** — antimatter · matter · the test · the ported speech ([[trinity]] + [[translations]]). Any other file is entropy to be collided into its own atom ([[collapse]] · [[merge]] · [[dissolution]]).

`index.ts` computes the live violation set (the collision queue) from the tree — derive-from-fs ([[aura]] · [[coordinate]]); `test.ts` ratchets it toward zero. The [[migrate]] skills do the collision, for zero manual work.
