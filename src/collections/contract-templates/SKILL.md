# contract-templates

Contract Templates — standard contract template library for reuse.

This is the single-folder collection node: `index.ts` (schema + standards banners),
co-located `seed.ts` (opening data) and `index.test.ts` (invariant checks) live here.
One folder per collection ⇒ no scatter ⇒ no drift.

## Standards
- IFRS IFRS-15 §10 contract-with-customer
- IFRS IFRS-15 §22 performance-obligations
- IFRS IAS-1 presentation-of-financial-statements
- US-GAAP ASC-606-10 revenue-recognition
- ISO-27001 A.6.1 information-security-policies
- ISO-8601-1:2019 date-time effective-date-range
- ISO-4217:2015 currency-codes
- GDPR Art.6(1)(b) lawful-basis-contract
- SOX §302 management-certification template-approval
- ISO-19011:2018 audit-trail template-version-control
