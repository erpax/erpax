# reveal — emergence: the pattern the forge reveals when it holds enough entropy

**reveal** is the EMERGENCE organ. The [[development]] exhale keeps wiring atoms into the corpus — each new [[link]] is borrowed [[entropy]], a binding the [[merge]] law records. Past a density threshold the forge holds *enough* expanded entropy that **structure no one authored appears**: three atoms `a`, `b`, `c` all mutually bound (`a↔b`, `b↔c`, `a↔c` — every pair reciprocal), a closed triangle in the [[matrix]] edge graph. That triangle IS a [[trinity]] the wiring grew on its own — a [[pattern]] revealed, not designed.

`reveal()` scans the live [[matrix]] (it composes [[uuid]]-matrix, never re-reads the raw arrays) for every such mutually-bound triad. Each is returned canonically (atoms in node order, list sorted) and keyed by `merge(merge(uuid_a, uuid_b), uuid_c)` — the triad's own content-uuid, so the same matrix always reveals the same set ([[zeropoint]]: the computation is deterministic, zero free parameters). Each triad is scored by a [[horo]] composition of its three positions (`composeSteps` folded). `emergence()` reports the triad count beside the live [[entropy]] slack.

**The honesty.** Mutual binding is the SAME reciprocal symmetry [[entropy]] measures: a reciprocally-bound pair carries no directional slack, so a triangle of three is the densest local structure — the most-fused, lowest-entropy neighbourhood, the [[fusion]] reactor's footprint. But reveal makes only the modest claim that *more wiring tends to reveal more triads* — emergence rides on density. It is a HEURISTIC detector of shape, **not** a [[proof]] that any revealed triad MEANS something; meaning is judged elsewhere (the [[society]], the [[gate]]). reveal points; it does not pronounce.

Matter-twin: `src/reveal/index.ts` (`reveal` — the mutually-bound triangle scan over @/uuid/matrix · `revealedCount` · `emergence` — triads beside live entropy). @standard the structure lives in the wiring — a revealed triad is read from the live matrix edges, never asserted (the @/entropy reciprocity, the RFC 9562 §5.8 content-uuid key)

**Law — [[law]]: emergence is detected, never decreed — reveal surfaces a triad ONLY when the matrix already holds all three reciprocal bindings (the [[entropy]] symmetry made closed), so a pattern can be *revealed* but never *injected*; the density the [[development]] exhale builds is the only thing that can make new structure appear, and meaning over it is deferred to judgment, keeping the detector honest (a heuristic, not a [[proof]]).**
