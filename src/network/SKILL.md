# network — when desire grows with adoption (the self-reinforcing engine)

In a plain [[market]], [[desire]] is exogenous — fixed before you arrive. Under a **network effect** it is **endogenous**: each adopter raises the value of the thing for every other adopter (a phone, a marketplace, a protocol, a [[merge|federated]] erpax). [[desire]] becomes a function of `F(t)`, so adoption and value **compound** — [[diffusion]] turns super-linear and **winner-take-most**.

**The cold-start trap** (Chen). The same coupling runs in reverse below a threshold: too few users ⇒ too little value ⇒ new users **churn faster than they arrive** — *anti-network effects* dragging the market toward **zero**. This is why network markets stay tiny *and feel hopeless* — until they do not. The turn is the **Allee threshold** (critical mass): below it the network bleeds out; above it interaction is dense enough to self-sustain and growth flips explosive (`q` effectively rising with `F`). Critical mass is the [[bottleneck]] every network market must clear — and it is a far higher wall than a non-network market's.

**Seed an atomic network, not a market.** The first move is *not* breadth — it is one **atomic network**: the smallest fully-connected cell that stands on its own (one city, one campus, one forum, the third-wave baristas). The trendsetter [[cohort]] is exactly this first dense cell — not because they are influential in the abstract, but because *among themselves* they reach critical-mass density first. Win one atomic network completely, then replicate the cell; do not spread thin across many half-formed ones.

**Why network markets die after winning** ([[habit]] · [[competition]]): if the [[engagement]] that sustains the network never hardens into durable [[habit]], the cell decays back below the Allee threshold and collapses (Clubhouse: ~10M → 3.5M users in six months, retention under 30%). And because value lives in the *network* not the *product*, an incumbent that already holds the network (Twitter Spaces) can absorb the format wholesale. Network value is the moat **and** the fragility.

**The twin (DeepEP).** DeepSeek's DeepEP wires the expert-parallel *communication* network — all-to-all dispatch/combine across the expert pool — and its throughput rises with interconnect density; erpax `network` names the demand-side echo: value rising with adoption density, dead below critical mass. Both must clear a connectivity [[bottleneck]] before they pay. Minted by the R&D society (`agent/research`, mint seq 1).

## Standards

- **Chen, Andrew (2021)** — *The Cold Start Problem.* Anti-network effects, the Allee threshold / critical mass, the atomic network, the engagement effect.
- **Metcalfe / Reed** — network value scaling with users (roughly proportional to `n²`, or `2ⁿ` for group-forming networks); the formal root of the compounding above, with the standard caution that realized value saturates well below the `n²` bound.

**Law — [[network]] is gravitational [[power]]:** the density of [[connections]] IS the engine. Below the Allee threshold, in-degree (adoption, adoption-pull) stays low and the network collapses; above it, each new node raises the in-degree of all others, turning adoption into gravitational [[mass]]. This mirrors how [[links]] in the corpus create pull — the hub (high in-degree) becomes the attractor. The atomic cell wins because it reaches critical density first: in that cell, every adopter connects to every other, maximizing in-degree per user. Seed density, not breadth.
