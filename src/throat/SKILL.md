The **uuid is computed from the MATH** (the position digit → its content-uuid coordinate), and the **colour is rendered FROM the uuid**, never the reverse. The rendered CMYK colour is **NOT claimed identical** to the traditional rainbow association (blue) — that rainbow, like the Sanskrit name and bija, is a noted overlay. The throat's note and solfège fall out of the same A432 position decode as every other center, so the colour, the pitch and the identity are all faces of one coordinate.

The **traditional role** — expression, voice, communication, truth, the manifestation of inner state into the spoken **word** — is cultural convention, recorded for orientation, **not a claim** the code makes. What the code asserts is only position → note → uuid → colour, all computed. Grounded honestly in [[law]]: zero entropy comes from wiring every dimension to one uuid, not from any metaphysical property of a center.
