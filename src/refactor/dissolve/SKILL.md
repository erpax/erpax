# dissolve — collapse prefixes into the single-word matrix

The deterministic migration the [[sequence]] address law implies — **location = word-path, no prefix**. Drops the grouping prefix (collections/services/…), splits camelCase/hyphen segments into nested single-word folders, applies the STRICT naming law (singular model · plural collection), remaps every `@/` import (depth-independent after [[normalize]] runs first), rewires the registry barrels. Same target word ⇒ an accountable collection (merge, never last-wins — a known gap: it skips by detected target-collisions only, not against pre-existing files).

Matter: `index.mjs` (`--apply` executes; default dry-run). Composes [[normalize]] · [[sequence]] · [[identity]] · [[merge]].
