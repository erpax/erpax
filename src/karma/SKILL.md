# karma — the entropy moral ledger

Karma is the **moral reading of entropy**. Creating order (↓entropy) is **good** karma; destroying order (↑entropy — waste, war) is **bad** ([[peace]]: building has positive karma, destroying none — the [[angel]] / archangel balance).
