# equity

The owners' residual — assets minus liabilities. In erpax readme accounting, every asset debit pairs `[[equity]]/[[balance]]` so the journal closes ([[balance]] · [[conservation]]).

**Law — [[law]]: equity is one accounting word, content-addressed; the same concept collapses to one atom in the corpus lattice.**
